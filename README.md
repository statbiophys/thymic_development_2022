## Reproduce Plots

In order to reproduce the plots:

1) Extract the file 'CA-lienar_models.tar' in this folder.

2) Install the Required Packages (see section below).

3) Run the corrispondent Jupyter notebook.

## Required Packages

1) Default pip install:
- `numpy`
- `pandas`
- `matplotlib`
- `scipy`
- `scikit-learn`
- `seaborn`
- `tensorflow`
- `sonia`
